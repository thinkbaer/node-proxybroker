== Issues


* CLI

** judge :proxy_ip :proxy_port +
  Ausführen eines Tests für die jeweilige IP + Port
** update [:task] +
  Aktualisierung der ProxyListe
** start +
  Starten des ProxyServers
** stop +
  Stoppen des ProxyServers
** print +
  CSV, JSON, Liste Ausgabe der aktuellen Liste
