### Entities

General parameters:

 * Filtering

 GET /entities?count>0&name=*foo*

 * Sorting

 GET /entities?sort=-count,+id

 * Paging with offset and limit

 GET /entities?offset=10&limit=10

 * Field selection with list of given fields

 GET /entities?fields=id,name
